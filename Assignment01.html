World Wide Web



General

The foundation of the WWW are three technologies:
  • uniform resource identifier/locator (URI/URL)
  • URI Syntax: [//[user[:password]@]host[:port]][/path][?query][#fragment]

  • HTML HyperTextMarkupLanguage

  • HTTP HyperText Tranfer Protocol


Webserver

A Webserver offers an webservice
but its primary function is to store, process and deliver web pages to clients
The communication between client and server takes place using the Hypertext Transfer Protocol (HTTP)


Webclient/browser

A browser is an client-side application for retrieving, presenting and traversing information with an webserver.
the browser interprets the data from the server. This is the reason why different webpages do not look the same on different browsers.
